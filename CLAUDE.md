## Guide to Claude
1. Write maintainable, scalable and modular code.
2. Make sure the code is easy to debug through relevant logs.
3. Make sure you encounter all edge cases and handle them professionaly.
4. Make sure you don't introduce any breaking changes or introduce new bugs.
5. Make sure UI/UX is concise and adhering to our glass morphism style in both light and dark mode.
6. Make sure to use ready made generic component to be more consistent in the UI.
7. Make sure to create todos and implement them perfectly covering all the requirements.
